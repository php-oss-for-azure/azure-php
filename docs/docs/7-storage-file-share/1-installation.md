`azure-oss/storage-file-share` is the Azure Files package for PHP.

## Requirements

- PHP 8.2+
- PHP extensions:
  - `curl`
  - `json`
  - `simplexml`

## Install With Composer

```bash
composer require azure-oss/storage-file-share
```
